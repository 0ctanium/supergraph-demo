# Supergraph Demo

## Managed Federation

```sh
# pull subgraph schemas with federation enrichments
make introspect

# publish subgraph schemas to a new federated graph superschema
make publish

```

## Local Supergraph Composition

```sh
# pull subgraph schemas with federation enrichments
make introspect

# build a supergraph config file
make config

# locally compose a supergraph
make compose

```
