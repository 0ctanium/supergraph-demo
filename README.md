# Supergraph Demo

## Managed Federation

```sh
# pull subgraph schemas with federation enrichments
make introspect

# publish subgraph schemas to a new federated graph superschema
make publish

```

## Local Supergraph Composition

```sh
# pull subgraph schemas with federation enrichments
make config

# publish subgraph schemas to a new federated graph superschema
make compose

```
